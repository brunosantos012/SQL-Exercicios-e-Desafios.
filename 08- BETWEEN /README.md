Neste módulo de SQL, aprendi a utilizar a cláusula BETWEEN, que facilita a criação de consultas mais limpas e organizadas.

Com o BETWEEN, é possível filtrar registros que estejam dentro de um intervalo específico de valores, evitando a necessidade de escrever múltiplas condições e tornando o código mais legível.

OBS: AS CONSULTAS DESSE MODULO FORAM TIRADAS DO BANCO DE DADOS: ContosoRetailDW.
