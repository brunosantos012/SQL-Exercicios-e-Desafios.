Neste módulo de SQL, aprendi a utilizar a cláusula IN para simplificar consultas que envolvem múltiplas condições.

Com o IN, é possível substituir várias condições utilizando OR, tornando o código mais limpo, legível e fácil de manter, além de facilitar a criação de filtros mais eficientes.
